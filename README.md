### **3. Termux Deployment**
Run the following commands in Termux:
```bash
apt update
apt upgrade
pkg update && pkg upgrade
pkg install bash
pkg install libwebp
pkg install git -y
pkg install nodejs -y 
pkg install ffmpeg -y 
pkg install wget
pkg install imagemagick -y
git clone https://github.com/Toxic1239/Queen-RiasV3_
cd Queen-RiasV3
yarn install
yarn start
```
